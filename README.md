# Our Love Story

## Project Overview
"Our Love Story" is a web-based interactive experience that takes you through a journey of cherished memories. It utilizes a captivating mix of HTML, CSS, and JavaScript, along with Tailwind CSS and Font Awesome, to create an engaging user interface. The application features a responsive design, beautiful animations, and seamless navigation through different screens depicting special moments in a relationship.

## Installation
To set up and run the project locally on your machine, follow these steps:

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate into the project directory:
   ```bash
   cd our-love-story
   ```

3. Open `index.html` in your preferred web browser:
   ```bash
   open index.html   # For macOS
   xdg-open index.html   # For Linux
   start index.html   # For Windows
   ```

## Usage
The application consists of multiple screens that users can navigate through to view memorable moments. Users can:

- Click on the buttons or swipe left/right (or use keyboard arrows) to navigate through the screens.
- Restart the journey at any time by clicking the "Volver a empezar" button.

### Keyboard Controls
- **Arrow Right or Space**: Navigate to the next memory.
- **Arrow Left**: Navigate to the previous memory.

### Touch Controls
- Swipe left or right to navigate between memories on touch-enabled devices.

## Features
- **Interactive Navigation**: Navigate through memories with buttons or swipe gestures.
- **Animated Transitions**: Smooth fading and floating animations enhance user experience.
- **Responsive Design**: Adapts seamlessly to various screen sizes and devices.
- **Beautiful Typography**: Utilizes custom Google Fonts for a unique look.

## Dependencies
This project relies on the following libraries:

- **Tailwind CSS**: A utility-first CSS framework for styling.
- **Font Awesome**: Provides icons and social logos for a visually appealing interface.

These dependencies are included via CDN links in the `index.html` file and do not require local installation.

## Project Structure
The project consists of the following files:
```
/our-love-story
├── index.html                  # The main HTML file containing the entire application structure and logic.
```

### Styles and Scripts
- **Inline CSS**: The styling is embedded in the `<head>` section of `index.html`.
- **JavaScript**: All interactive scripts are included in the `<script>` tag at the end of `index.html`.
